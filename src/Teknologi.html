1.Beskriv en computers hardware.

SVAR:

En computer består af mange forskellige komponenter; en PSU (Power Supply Unit),
en, GPU (Graphics Processing Unit), en CPU (Central Processing Unit), en Motherboard,
køleranlæg (Fans) og Kabinettet.

2. Beskriv Azure Cloud og hvad du kan bruge den til.

SVAR:

Microsoft Azure Cloud er en form for hub, hvor man kan sætte servers, databaser
VM (Virtual Machines) og mange flere services gennem en skyen. Man kan
tilpasse disse services, så det passer den opgave man gerne vil bruge dem til.

3. Hvilke typer harddiske findes i en computer og hvad er forskellen på dem?

SVAR:

Der er 3 forskellige HDDs (Hard Drives), SATA, SSD og NVMe.
SATA (Serial Advanced Technology Attachment), er nok det billigste
form for opbevaring af filer. Meget plads for pengene, men ikke så brugbart
til mindre medier, såsom laptops, da SATAer fylder meget.
SSD (Solid State Drive) er mindre og hurtigere, men dyrere end SATA.
NVMe (Non-Volatile Memory Express) er det hurtigste form for dataopbevaring,
den er en form for SSD, som sættes på selve motherboardet.
